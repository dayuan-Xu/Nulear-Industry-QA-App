# Nulear-Industry-QA-App
核工业专业知识问答应用

[LangChian官网](https://python.langchain.com/docs/how_to/)

# 进度安排
- 4-5月：在本地实现对常见类型文件的解析和存储，如TXT文件、PDF文件、DOCX文件、PPTX文件，之后实现控制台问答。
- 6-7月：利用LLM从用户输入构建结构化查询，同时实现数据导入、数据检索进度可视化，检索结果可追溯，令牌使用量可视化。
- 8-9月：学习Streamlit前端框架，将问答模型部署为Web服务，设计必备的用户界面，包含知识库管理、模型选择、问答（对话）界面管理。
- 10-12月：实现Web界面的各种功能。
- 次年1月-2月：增加文件解析范围，比如CSV文件、HTML文件、MD文件。
- 次年3月：增加代理功能等优化工作。
